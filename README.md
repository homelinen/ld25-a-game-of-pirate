# A Game of Pirates #

My submission for the 25th Ludum Dare Competition with the theme: "You
are the Villain"

## Controls ##

* Movement: `Arrow Keys`
* Fire Canon: `space` 

## Install ##

Windows: [a-game-of-pirates.exe](http://calumgilchrist.co.uk/ludum-dare/a-game-of-pirates.exe)

### Run From Source ###

    $ bundle install --without development
    $ ruby pirates.rb
